#variables

`$PATH` is an **environment variable** that tells the shell where to search for executable commands.

When we run:

```bash
echo $PATH
```

It prints:

```bash
/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/root/bin
```

Each directory is separated by a colon `:`.

If you type `ls`, bash checks:

1. `/usr/local/sbin/ls`
2. `/usr/local/bin/ls`
3. `/usr/sbin/ls`
4. `/usr/bin/ls`
5. `/root/bin/ls`

The first match it finds, it executes.
