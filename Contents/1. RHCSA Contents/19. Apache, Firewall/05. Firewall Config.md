#firewall #configuration 

## **Firewall Config**

To see firewall config (for current zone), we need to run this command:

```bash
firewall-cmd --list-all
```

Here, the main parts are `services` and `ports`.

---
## **Adding Ports to Firewall**

We can add some ports to firewall, so, they will not blocked by firewall anymore.

The commands:
```bash
firewall-cmd --add-port=80/tcp
firewall-cmd --permanent --add-port=80/tcp
```

The first command here adds the port to firewall temporary, so, after reload the change will disapper. The second command affects after the reboot. Mostly, we need to run both.

To remove the port:
```bash
firewall-cmd --remove-port=80/tcp
firewall-cmd --permanent --remove-port=80/tcp
```

---
## **Adding Services to Firewall**

We can also add some services to firewall, so, they will not blocked by firewall anymore.

The commands:
```bash
firewall-cmd --add-service=ntp
firewall-cmd --permanent --add-service=ntp
```

The first command here adds the service to firewall temporary, so, after reload the change will disapper. The second command affects after the reboot. Mostly, we need to run both.

To remove the service:
```bash
firewall-cmd --remove-service=ntp
firewall-cmd --permanent --remove-service=ntp
```
