## **What is Swap?**

Swap memory, also known as swap space, is a portion of a computer's hard drive or SSD that the operating system uses to store inactive data from RAM when it runs out of available memory. This helps prevent system slowdowns or crashes by allowing the OS to continue functioning even when physical RAM is full.

To look at the current swap memory, we need to run this command:

```bash
free -mh
```

It will show total, used, free RAM and swap memory.

## **How to create/increase swap memory?**

It is a very easy process. First of all, we need to have a [[02. Partitioning and Its Types|partition]] to give it to swap. We have learned how to divide a [[01. Introduction to Disk|disk]] to partitions. 

Now we need to write [[04. Filesystem|filesystem]] on this partition. However the filesystem for swap is different.

```bash
mkswap /dev/nvme0n2p2
```

After adding filesystem, we need to enable this memory:

```bash
swapon /dev/nvme0n2p2
```
