#introduction #fstab #theory #advance

`/etc/fstab` is a configuration file that tells Linux **which [[01. Introduction to fstab file|filesystems]] to [[05. Mounting|mount]] automatically at boot** and where to mount them.

Each line represents one mount entry with 6 fields:

```
/dev/vg0/lv_data  /data  xfs  defaults  0  0
device            mount  fs   options  dump pass
```

Without an entry here, your mount disappears after reboot and you'd have to mount it manually every time. That's why after creating and mounting a new filesystem, adding it to fstab is always the final step.
