#scripting 
## **Positional Parameters**

```bash
./script.sh Ravan 25
```

Inside the script:

```
$0  → script name itself (./script.sh)
$1  → first argument  (Ravan)
$2  → second argument (25)
$3  → third... and so on
```

---

Simple example:

```bash
#!/bin/bash
echo "Name: $1"
echo "Age: $2"
```

```bash
./script.sh Ravan 25
# Output:
# Name: Ravan
# Age: 25
```

---

**A few related special variables worth knowing:**

```
$#  → total number of arguments passed
$@  → all arguments as separate items
$?  → exit code of the last command (0=success, non-zero=fail)
$$  → PID of the current script
```

`$?` is especially useful — you'll use it constantly to check if a command succeeded:

```bash
systemctl start nginx
if [[ $? -ne 0 ]]; then
    echo "Failed to start nginx"
fi
```
