#stratis #commands 

First we install stratis service and command-line utilities:

```bash
dnf install stratisd stratis-cli
```

Then we start the `stratisd` service and enable it to lauch at boot:

```bash
systemctl enable --now stratisd
```

Then we verify whether the service is enabled and running:

```bash
systemctl status stratisd
```

We need to erase any file system, partition table and RAID signatures that exist on each block device that you want to use in the Stratis pool:

```bash
wipefs --all <BLOCK_DEVICE>
# The BLOCK_DEVICE value is the path to the block device; for example, `/dev/sdb`.
```

Then we create a pool:

```bash
stratis pool create <POOL_NAME> <BLOCK_DEVICE>
```

Verify that the pool is created:

```bash
stratis pool list
```

Then we create a filesystem in the pool:

```bash
stratis filesystem create --size <SIZE> <POOL_NAME> <FS_NAME>
```

When we mount its filesystem, we need to write this to [[02. Syntax of fstab file|fstab]] file:

```bash
/dev/stratis/<POOL_NAME>/<FS_NAME> <MOUNTPOINT> xfs defaults,x-systemd.requires=stratis-fstab-setup@pool-uuid.service,x-systemd.after=stratis-fstab-setup@pool-uuid.service dump-value fsck_value
```

It supports only `xfs` as a filesystem.
