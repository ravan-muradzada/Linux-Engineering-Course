#commands #alias

## **`alias`**

This commands help us to create shortcuts for other (mostly huge) commands. So instead of writing huge commands which are difficult to remember, we can use alias, so it is easy to remember and takes less time to write.

Syntax:
```bash
alias name='command'
```

Example:
```bash
alias cls='clear'
alias gs='git status'
```

If we write just `alias` command in terminal, it will show us all aliases.

----

## **`unalias`**

This command removes an alias you no longer want.

Syntax:
```bash
unalias name
```

Example:
```bash
unalias cls
unalias gs
```

Remove all alias at once:
```bash
unalias -a
```
