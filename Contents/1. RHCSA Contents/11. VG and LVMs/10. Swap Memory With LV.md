We can define [[05. Logical Volume Management|LV]] as a swap memory, so that makes it resizable. The process is very similar, however we need to define its filesystem as swap:

```bash
mkswap /dev/vg_name/lv_name
```

Then we need to add it to [[01. Introduction to fstab file|fstab]] file and write it like that:

```bash
UUID=<UUID>   none   swap   defaults 0 0
```

- In the third place, we write `none` or `swap`.
- We write `swap` as filesystem.

Then we need to 'mount' it (it is not actual mount):

```bash
swapon -a
```

We can look at swap memory with `free -mh`.
