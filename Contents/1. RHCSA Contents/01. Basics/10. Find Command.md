## **What is `find` Command?**

`find` command helps us to find specific file(s) in the system. It can search in full system or a specific directory. We use wildcards for `find` command. It has very powerfull sides like doing operations on the output of the search.

Basic syntax:
```bash
find . -type <TYPE> -name <NAME> -perm <NUM>
```
- After the command itself we need to specify the searching area; `.` means the current directory, `/` means full system.
- If we want to specifically find one type of documents, we use -type option. (Optional)
- If we want to search by names, we need to use -name option. (Optional) Here we use wildcards.
- If we want to search by permissions, we use `-perm` like `-perm 543`.
