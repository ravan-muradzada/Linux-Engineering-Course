#permissions #commands 

`umask` command is used to set default permissions for files and directories. Default value for `umask` is `0022`. When we run `umask` command directly, we can its value.

The default permissions of files are calculated from `666`, it is `777` for directories. We subtract `umask` value from these and find the default permissions for them.

- Directories: `777` - `022` = `755` → `rwx r-x r-x`
- Regular files: `666` - `022` = `644` → `rw- r-- r--`

----

We can update `umask` value:

```bash
umask <VALUE>
```

However it changes temporarily.

To update `umask` value permenantly, we need to open `/etc/bashrc` [[06. Bash Config Files|file]] and change here.
