#package-managers #commands

## **`yum grouplist`**

This command shows **software groups** available and installed.

```bash
yum grouplist
```

But by default, it may **not show all groups**, only the ones enabled or installed. To see **all groups**, use:

```bash
yum group list --hidden
```


> [!Info] GUI Version
> If we want to switch to GUI version, first we need to download the group:
> `yum group install "Server with GUI"`
> After downloading it we just enable this [[05. Systemd Targets|target]]:
> `systemctl isolate graphical.target`
> To switch back to minimal version:
> `systemctl isolate multi-user.target`

---

## **`yum groupinstall <GROUP_NAME>`**

This command installs a **software group** (a set of packages grouped for a purpose, like “Development Tools”).

```bash
yum groupinstall "Development Tools"
```
