#commands #shell

In Linux (bash shell), commands are either **internal (built-in)** or **external**.

Internal commands are built into the shell itself. They are part of `bash`. They do not exist as separate files in `/bin` or `/usr/bin`. The shell executes them directly without starting a new process.

So it takes more time to run external commands, however after the first call they are kept in cache and it increases their speed.

To be able to determine whether any command is internal or external, we need to use `type` command:

Example:

```bash
type pwd
type tar
```

- For intenal commands like `pwd`, it will show: `pwd is a shell builtin`.
- For external commands like `tar`, it will show: `tar is /usr/bin/tar`
