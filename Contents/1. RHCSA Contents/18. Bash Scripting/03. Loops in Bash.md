#scripting 
## **For loop** 

When we know exactly what to iterate over:

```bash
for i in 1 2 3 4 5; do
    echo "Number: $i"
done
```

Or with a range:

```bash
for i in {1..5}; do
    echo "Number: $i"
done
```

Or C-style (like most programming languages):

```bash
for (( i=1; i<=5; i++ )); do
    echo "Number: $i"
done
```

---
## **While loop**

It runs as long as condition is true:

```bash
count=1
while [[ $count -le 5 ]]; do
    echo "Count: $count"
    (( count++ ))
done
```

Stops when condition becomes false. If condition starts false, it never runs.

---
## **Until loop**
It is opposite of while, runs as long as condition is false:

```bash
count=1
until [[ $count -gt 5 ]]; do
    echo "Count: $count"
    (( count++ ))
done
```

Same result as above, just opposite logic.
