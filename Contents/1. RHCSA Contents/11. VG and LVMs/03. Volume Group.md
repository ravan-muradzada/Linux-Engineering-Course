## **What is VG?**

**Volume Group** ŌĆö it's the pool you create by combining one or more PVs together. The VG doesn't care how many physical disks are underneath ŌĆö it just sees one combined pool of space.

---

## **Commands**

We use `vgcreate` command to create a new volume group:

```bash
vgcreate <VG_NAME> <PARTITION_NAME>
```

We can give any name for our vg. 

We can use `vgs` command to display information about all volume groups:

```bash
vgs
```


We have another method which also shows information about all VGs, but more detailed:

```bash
vgdisplay
```

---
## **About Physical Volume**

Before creating VG, we need to create [[02. Physical Volumes|PV]]. However, `vgcreate` command created proper PV for us automatically. So, we do not need to use `pvcreate` to create a new PV manually.
