#commands 

To be able to see the command history, we have a specific command:

```bash
history
```

To see last 10 commands, we can use combinations like:

```bash
history | tail -n 10
```

In the output of `history` command, it always show the command number, so if needed we can run specifically that number again like that:
```bash
!82
```

To run the last command, we need to run `!!`.

---

To delete history of current shell:

```bash
history -c
```
