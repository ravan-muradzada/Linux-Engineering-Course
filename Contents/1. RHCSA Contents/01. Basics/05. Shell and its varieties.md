## **What is Shell?**

In Linux, a **shell** is a command-line interface that allows users to interact with the OS by typing commands, running programs, and writing scripts. The most famous Linux shells are **Bash (Bourne Again Shell)**, **Sh (Bourne Shell)**, **Ksh (Korn Shell)**, **Csh (C Shell)**, and **Zsh (Z Shell)**. **Bash** is default shell in Linux.

---

## **Difference between SH and BASH**

Beforehand **sh** is default shell in linux, but now it is **bash**. The main difference between **sh** and **bash** is that Bash is an improved, backward-compatible version of sh with extra features like command history, tab completion, better scripting capabilities, and improved variables handling. 

We use **Bash** today instead of sh because modern Linux systems need these enhanced features for easier scripting, interactive use, and better compatibility with new software. The shift from Sh to Bash happened to provide more functionality and convenience while keeping sh scripts still runnable.
