## **What is PV?**

**Physical Volume (PV)**  ŌĆö it's simply a disk or partition that has been initialized for LVM use with `pvcreate`. It's just LVM's way of saying "I know about this storage device and it's ready to be added to a Volume Group."

---
## **Commands**

To create a new physical volume, we need to use:

```bash
pvcreate <PARTITION_NAME>
```

We write here the name of partition like `/dev/nvme0n2p1`.


We can use `pvs` command to display information about all physical volumes.
```bash
PV            VG   Fmt  Attr PSize  PFree
/dev/nvme0n1p3 rhel lvm2 a--   8.41g     0
/dev/nvme0n2p1      lvm2 ---  <6.00g <6.00g
```

It gives that kind of output.
