#commands 

Every command in Linux returns an exit code whether it succeedes or fails. If a command is successfull, 0 is returned as an exit code. However if there is an error, depending on the error type the exit code between 1 and 255 is returned.

To see the exit code for the last run command:

```bash
echo $?
```

We can search and find the meaning of each exit code from the internet.
