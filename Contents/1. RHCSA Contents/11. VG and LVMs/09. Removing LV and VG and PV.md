## **Removing LV**

To remove [[05. Logical Volume Management|LV]], first we need to unmount it. So we delete its mount info in `/etc/fstab` and run `umount /mountpoint`.

Then we use this command:

```bash
lvremove <LV_PATH>
```

Path needs to be something like `/dev/vg01/lv01`.


---

## **Removing VG**

To remove [[03. Volume Group|VG]], we use this command:

```bash
vgremove <VG_PATH>
```

Path needs to be something like `/dev/vg01`.
